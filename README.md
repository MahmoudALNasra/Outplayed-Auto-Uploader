# Outplayed Auto-Uploader for YouTube

This tool automatically scans your Outplayed recordings folder, formats the file names (Game + Date + Time), and uploads any new clips directly to your YouTube channel. 

It uses a memory file so it will never upload the same video twice, even if you restart your PC.

## Features
- **Auto-Formatting:** Turns `Valorant_05-10-2025_22-13-59-550.mp4` into `Valorant Gameplay - 05-10-2025 22:13:59`.
- **Duplicate Prevention:** Keeps a log of uploaded files.
- **Run on Startup:** Can be configured to run silently in the background every time you turn on your computer.

Please see `INSTALLATION.md` for instructions on how to set this up on your PC.
